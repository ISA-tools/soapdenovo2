Overview

Data Models

The nanopublications for the SOAPdenovo2 use case were built using a template for OntoMaton, which combines Google Spreadsheets with services for ontology lookup and annotation relying on NCBO Bioportal and Linked Open Vocabularies (this last one was added in the recent version of OntoMaton). The spreadsheet with the Nanopublications is here.
